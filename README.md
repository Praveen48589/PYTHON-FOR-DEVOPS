# Project : DEVOPS-UTILITIES-API

## Aim

This project aims to deliver Production grade
Utilities APIs to Internal team:

- AWS Resources APIs
- Metrics
- Log Analysis

## Usage

step 1 : 'git clone <>'

### Setup Python Environment

step 2 : python3.14 -m env venv

step 3 : source venv/bin/activate

### Install Requirements
step 4 : pip install -r requirements.txt

### Run Application
step 5 : main.py
